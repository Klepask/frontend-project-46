# «Вычислитель отличий»

Вычислитель отличий – программа, определяющая разницу между двумя структурами данных. Поддерживает форматы: yaml, json.

### Установка

Необходима установка node.js
```
$ git clone https://github.com/Klepask/frontend-project-46
```
